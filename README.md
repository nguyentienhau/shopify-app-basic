# Shopify App Basic

## Install Shopify Dependencies

    npm install

## Install Frontend Dependencies

    cd frontend
    npm install

## Install Backend Dependencies

    cd backend
    npm install

## Create shopify.app.toml File

    npm run config:link

## Sync Scopes Setup With App By Release New Version

    npm run deploy

## Pull Env File

    npm run env:pull

## Run App

    npm run dev
